# Wudase Mariam Memorization Tracker

A personalized tool to memorize the daily prayers of Wudase Mariam.

## Features
- **Daily Restriction:** System forces recall of all pending paragraphs.
- **SRS Ratings:** Grade your recall (Forgot, Hard, Easy).
- **Mastery Analytics:** Track progress with "Needs Work" vs "Mastered" indicators.
- **Streak Tracker:** Keep your spiritual discipline consistent with daily counts.

## How to use
1. Upload your prayer paragraphs via the /upload/ endpoint.
2. Visit /daily-recall/ to start your daily cycle.
3. Check the dashboard to see your 30-day progress and mastery status.
